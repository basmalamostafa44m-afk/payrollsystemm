# Corporate payroll system management🏢
## Overview
This a simple python project that calculates employee's
bonus,tax and net salary and display them to user

## How it works
1.The user enter:
- Name
- Department
- Base salary
- Performing rate

2.The program calculate
- Bonus based on the employee's rate
- gross salary by adding the bonus to the base salary
- Tax based on gross salary
- Net salary by subtracting taxes from gross salary

3.program display a whole payroll statement to the user
## installation
```bash
git clone https://github.com/basmalamostafa44m-afk/payrollsystemm.git
cd payrollsystemm
python main.py
```
